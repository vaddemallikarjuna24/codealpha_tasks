# Sentiment Analysis using Python

## Objective
Analyze customer reviews and classify them as Positive, Negative, or Neutral.

## Tools Used
- Python
- Pandas
- TextBlob

## Dataset
Customer Reviews Dataset

## Features
- Read customer reviews
- Analyze sentiment
- Classify reviews
- Save output to CSV

## Output
Creates a file named `sentiment_output.csv` containing the review and its sentiment.
